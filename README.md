# 学习之路

## 项目简介

“学习之路”是 Aeve 的个人 AI 学习记录与复利工程网站。

网站用于记录从零开始学习 Codex、Claude Skills、AI 工作流和 AI 产品构建的过程，并保存可以重复使用的提示词、模板和方法。

## 当前版本

第一版 MVP。

当前版本采用原生 HTML、CSS 和少量预留 JavaScript，不使用框架、构建工具或外部依赖。

## 已完成功能

- 首页：展示网站介绍、当前学习方向、每日学习预览、复利工程预览、当前状态和下一步计划。
- 每日学习页：展示学习记录列表和第 1 天完整学习记录。
- 复利工程页：展示三条复利内容的列表和完整详情。
- 三个页面使用统一的顶部导航和页脚。
- 首页、学习记录和复利内容之间可以正常跳转。
- 详情内容可以通过页面内部锚点定位，并返回列表或首页。
- 已添加统一的基础样式和手机端适配。
- 页面可以在不使用后台和数据库的情况下直接打开。

## 如何在本地打开

本项目不需要安装依赖，也不需要执行构建命令。

1. 下载或解压项目文件夹。
2. 双击项目文件夹中的 `index.html`，即可在浏览器中打开首页。
3. 使用页面顶部导航访问首页、每日学习和复利工程。

也可以使用任意静态文件服务器打开项目目录，但第一版不依赖该方式。

## 本次学习收获

1. 写代码前先完成 PRD 和技术设计，可以让开发目标更清楚。
2. 把任务拆成基础结构、页面内容、统一样式和最终验收等小步骤，更容易检查和完成。
3. 每日学习记录负责保存学习过程，复利工程负责保存能够再次使用的成果。
4. 第一版只保留核心浏览流程，可以避免项目一开始变得过大、过复杂。
5. 使用统一结构和固定模板，可以降低后续更新与维护的难度。
